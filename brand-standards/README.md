# County of Marin Design Standards Skill Package

This package contains a reusable AI Skill and sub-skills for applying County of Marin visual identity and design standards to communications, print, digital, social, video, email, and template work.

Accessibility requirements are intentionally excluded from this package because they are handled by separate skills.

## Files

- `SKILL.md` — core routing, workflow, boundaries, and response patterns.
- `subskills/logo-identity.md` — logo use, identity hierarchy, placement, co-branding, web footer, video, social avatars.
- `subskills/color-typography.md` — color palette, fonts, hierarchy, charts, print and digital typography.
- `subskills/templates-media.md` — email signatures, slides, reports, social media, video graphics, forms, and reusable templates.
- `subskills/review-checklist.md` — design-standard compliance review checklist.
- `reference/source-notes.md` — source basis, exclusions, and limitations.

## Intended use

Use this package when an AI assistant needs to:

- Create a County-styled slide, document, email signature, chart, report, one-pager, social graphic, video title card, or web visual component.
- Check logo, color, typography, layout, or template compliance.
- Convert content into a County-appropriate visual format.
- Flag missing official assets or required brand/department review.

## Not included

This package does not include:

- Accessibility review.
- WCAG conformance requirements.
- PDF remediation.
- Screen reader, keyboard, alt text, caption, reading-order, heading-structure, or focus-state checks.
- Official logo artwork or internal County templates.
- Legal, privacy, records, procurement, or final brand approval.

Use the separate accessibility skills for accessibility-specific review.
