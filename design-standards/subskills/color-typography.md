# Color and Typography Sub-skill

Use this sub-skill for County of Marin colors, fonts, type hierarchy, data visualization, print collateral, web approximations, and slide/document styling.

This sub-skill covers visual design standards only. Do not include accessibility requirements or WCAG checks here.

## Color principles

Use the County palette from the published Identity Style Guide as the controlling visual reference. When exact color values are needed, use the official guide or a supplied brand file rather than estimating from screenshots.

Apply color as follows:

- Use County palette colors for identity, structure, emphasis, and wayfinding.
- Prefer restrained use of accent colors.
- Avoid arbitrary colors that are not in the County palette unless the asset has a documented campaign or program-specific palette.
- Maintain a clear hierarchy between headings, body text, labels, captions, and calls to action.
- Treat color as a support for meaning, not the only way meaning is expressed.

## Typography standards

Known public guide references:

- Futura appears in official logo/logotype artwork.
- Arial Regular is preferred for general print body copy.
- Georgia may be used as a serif option when needed.
- Verdana is identified as the website font in the guide.

Use these rules:

- Do not substitute or alter the typography inside official logo artwork.
- For routine documents and print collateral, default to Arial unless a template specifies otherwise.
- Use Georgia sparingly for formal serif treatments.
- Use Verdana when matching the published website standard is required.
- For modern digital systems where Verdana is not the platform default, preserve County hierarchy through scale, weight, line-height, and spacing.

## Type hierarchy

For most County materials:

- Use one clear primary title.
- Use short section headings.
- Keep body text visually subordinate to headings.
- Keep labels, captions, dates, and source notes consistent.
- Avoid excessive bold, all caps, underlining, drop shadows, outlines, and decorative effects.
- Maintain generous margins and predictable spacing.

## Charts and diagrams

For charts, dashboards, maps, and diagrams:

- Use County palette colors when feasible.
- Use the fewest colors needed to distinguish categories.
- Label directly where practical.
- Use consistent units, axes, legends, and captions.
- Avoid heavy gradients, 3D effects, texture fills, and decorative chartjunk.
- Use neutral supporting colors for grids, borders, and backgrounds.
- Make the main takeaway visually prominent.

## Layout guidance

- Favor clean grids, strong alignment, and visible hierarchy.
- Use whitespace to separate sections rather than relying on boxes around everything.
- Keep line lengths comfortable for the medium.
- Use repeated spacing and sizing tokens across a template.
- Avoid mixing too many type sizes or weights.

## Review checklist

Ask:

- Are colors drawn from the County palette or an approved campaign palette?
- Are fonts consistent with County guidance or the supplied template?
- Is the hierarchy clear at a glance?
- Are charts and diagrams visually restrained and readable?
- Are decorative effects avoided?
- Are any exact brand values missing and therefore unable to be verified?

## Response pattern

When reviewing color and typography, report:

1. What matches County guidance.
2. What is noncompliant or unverifiable.
3. Specific replacements, such as font family, heading scale, palette role, or chart treatment.
4. Whether exact official color values or templates are needed.
