# Inclusive Language Skill Package

This package provides a core skill and subskills for reviewing and creating respectful, resident-centered language.

## Files

- `SKILL.md` — core inclusive-language workflow and routing rules.
- `subskills/disability-language.md` — disability, access, accommodations, and ableist wording.
- `subskills/race-ethnicity-national-origin.md` — race, ethnicity, immigration, language access, and cultural identifiers.
- `subskills/gender-family-identity.md` — gender, pronouns, families, relationships, caregiving, and household assumptions.
- `subskills/age-and-income-sensitive-language.md` — age, income, benefits, fees, poverty, housing status, and financial hardship.
- `subskills/trauma-informed-language.md` — violence, crisis, disasters, grief, behavioral health, and harm-related content.
- `subskills/public-safety-and-health-language.md` — emergency, safety, health, enforcement, courts, and hazard communications.

## Scope

Use this package for editorial review, drafting support, and language risk identification. It does not replace legal, HR, clinical, privacy, civil rights, public health, law enforcement, emergency management, or community review.
