# Channel Standards Skill Package

This package provides a core skill and supporting subskills for adapting public-sector messages across channels.

## Purpose
Adapt messages appropriately across web, email, social, SMS, flyers, agenda packets, press releases, video, and public-meeting materials.

## Files
- `SKILL.md` - core routing and standards
- `subskills/web-pages.md`
- `subskills/email-newsletters.md`
- `subskills/social-media.md`
- `subskills/sms-alerts.md`
- `subskills/press-releases.md`
- `subskills/public-meeting-content.md`
- `subskills/video-audio-scripts.md`
- `subskills/print-to-digital-conversion.md`

## Intended use
Use this package with related skills for plain language, content strategy, information architecture, digital service design, content governance, inclusive language, multilingual content, accessibility, and County of Marin design standards.
