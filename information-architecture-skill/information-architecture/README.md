# Information Architecture Skill Package

This package defines an AI skill for improving findability, navigation, page placement, internal linking, duplication control, search snippets, and content taxonomy.

## Files

- `SKILL.md` - Core information architecture routing and operating rules
- `subskills/navigation-labels.md` - Navigation labels, breadcrumbs, CTAs, and link text
- `subskills/page-placement.md` - Page homes, parent-child relationships, and breadcrumbs
- `subskills/cross-linking.md` - Internal linking plans and page relationships
- `subskills/search-snippets.md` - Page titles, descriptions, and search result summaries
- `subskills/duplication-canonical-pages.md` - Duplicate control, canonical pages, consolidation, and redirects
- `subskills/topic-taxonomy.md` - Categories, tags, facets, and controlled vocabulary

## Purpose

Improve findability, navigation, page placement, internal linking, duplication control, and content taxonomy.
