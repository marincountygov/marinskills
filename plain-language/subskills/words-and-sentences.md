# Words and Sentences Sub-skill

Use this sub-skill when text is wordy, dense, legalistic, abstract, full of jargon, or hard to parse sentence by sentence.

## Main rules

- Prefer familiar words over unusual or formal words.
- Use the technical term only when the audience needs it.
- Keep one main idea in each sentence.
- Keep subject, verb, and object close together.
- Avoid noun strings and hidden verbs.
- Use consistent terms for the same concept.

## Edit patterns

### Replace hidden verbs

- make a payment -> pay
- conduct a review -> review
- provide assistance -> help
- make a determination -> decide
- submit an application -> apply, or send an application if a physical submission is required

### Break noun strings

When three or more nouns stack together, unpack the relationship.

- "program eligibility verification process" -> "how we verify whether you qualify for the program"
- "permit renewal fee notice" -> "notice about your permit renewal fee"

### Reduce abstractions

Prefer concrete actors, actions, dates, documents, and outcomes.

- "Noncompliance may result in adverse action" -> "If you do not follow these rules, we may deny your application."

### Use consistent terms

Do not alternate between resident, applicant, customer, participant, beneficiary, and client unless those categories are meaningfully different.

## Common replacements

- a number of -> some
- additional -> more
- adjacent to -> next to
- advise -> tell or recommend
- approximately -> about
- assistance -> help
- commence -> start
- comply with -> follow
- consequently -> so
- currently -> now, or omit
- demonstrate -> show
- determine -> decide or find out
- disseminate -> send or share
- due to the fact that -> because
- ensure -> make sure
- facilitate -> help
- frequently -> often
- in accordance with -> under, by, following, or according to
- in the event that -> if
- prior to -> before
- purchase -> buy
- regarding -> about
- request -> ask
- retain -> keep
- terminate -> end
- utilize -> use

## Sentence checks

Flag sentences that:

- Are longer than needed.
- Contain multiple conditions, exceptions, or embedded clauses.
- Start with a long introductory clause before the main point.
- Use passive voice without a reason.
- Hide the actor.
- Force the reader to remember too much before reaching the verb.
