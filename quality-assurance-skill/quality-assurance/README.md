# Quality Assurance Skill Package

This package provides a publication-readiness QA framework for public content and digital services. It coordinates review across brand, accessibility, plain language, governance, channel fit, information architecture, content strategy, service design, forms/documents, multilingual content, inclusive language, SEO/search, and data visualization.

## Files

- `SKILL.md` - Core skill and review workflow.
- `subskills/prepublication-checklist.md` - Final prepublication checklist.
- `subskills/cross-skill-review.md` - Broad review across related standards.
- `subskills/regression-review.md` - Review after changes, migration, or fixes.
- `subskills/manual-testing-plan.md` - Manual testing structure and test cases.
- `subskills/issue-severity-and-triage.md` - Severity model and prioritization.
- `subskills/publication-readiness.md` - Final publish/hold/escalate recommendation.

## Intended use

Use this skill when a public-facing asset needs structured QA before release or after revision. It does not replace specialist approval from legal, accessibility, privacy, procurement, public records, emergency management, or designated content owners.
