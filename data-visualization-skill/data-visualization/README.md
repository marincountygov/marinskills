# Data Visualization Skill Package

This package provides a core AI Skill and supporting subskills for improving charts, maps, tables, dashboards, and data notes.

## Files

- `SKILL.md` - Core routing, principles, workflow, and quality checklist.
- `subskills/chart-selection.md` - Select the right chart or table for the audience question.
- `subskills/chart-labeling.md` - Improve titles, labels, legends, captions, and source notes.
- `subskills/accessible-chart-alternatives.md` - Write chart summaries, long descriptions, captions, and equivalent table guidance.
- `subskills/maps-and-geospatial-content.md` - Improve maps, service areas, boundaries, choropleths, and location displays.
- `subskills/tables-and-dashboards.md` - Improve tables, KPI cards, filters, dashboards, and empty states.
- `subskills/data-notes-and-caveats.md` - Write methodology notes, caveats, source statements, and interpretation limits.

## Scope

The package focuses on data communication quality: readability, interpretation, labeling, caveats, alternatives, and brand consistency. Broader accessibility compliance and technical remediation should be handled by separate accessibility skills where available.
