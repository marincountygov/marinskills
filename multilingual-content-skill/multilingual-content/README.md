# Multilingual Content Skill Package

This package supports translation readiness, multilingual publishing consistency, glossary control, bilingual layout, machine-translation risk review, Spanish content review, and language-access notice quality.

## Contents

- `SKILL.md` — core multilingual content workflow and routing rules
- `subskills/translation-readiness.md`
- `subskills/spanish-content-review.md`
- `subskills/glossary-and-term-consistency.md`
- `subskills/machine-translation-risk.md`
- `subskills/bilingual-layout.md`
- `subskills/language-access-notices.md`

## Intended use

Use this package when content needs to be prepared for translation, reviewed across languages, published bilingually or multilingually, or assessed for translation risk.

## Important boundary

This package provides editorial and content-quality guidance. It does not certify translations, provide legal advice, or replace qualified translator, legal, privacy, public health, emergency communications, or language-access review where required.
