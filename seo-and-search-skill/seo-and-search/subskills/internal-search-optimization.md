# Internal Search Optimization

## Purpose
Improve results in the County website's internal search by aligning page content, labels, metadata, and synonyms with resident intent.

## Use when
- Residents cannot find known content through site search.
- Search results return duplicate or irrelevant pages.
- A page needs a better title, summary, or keyword configuration.
- Migrating or consolidating pages.
- Creating landing pages for high-volume topics.

## Principles

1. Optimize the destination, not just search configuration.
   - A page with unclear purpose will perform poorly even with added keywords.

2. Make result titles scannable.
   - Internal search users often scan quickly.
   - Avoid titles that begin with department names unless the department is the search target.

3. Use summaries that identify the task.
   - Internal result snippets should tell users what they can do on the page.

4. Reduce duplicate competition.
   - Multiple similar pages can split relevance and confuse users.
   - Recommend canonical pages and redirects when appropriate.

5. Build synonym support for resident language.
   - Synonym configuration should connect common terms to the right canonical page.

## Internal search review

For each target query, identify:

- Query term
- Likely resident intent
- Best destination page
- Current result issue
- Recommended title or metadata change
- Recommended synonym
- Recommended redirect or consolidation, if needed

## Common problems

- Department pages ranking above task pages.
- PDF documents ranking above HTML service pages.
- Old pages competing with current pages.
- Acronyms blocking discovery.
- Titles that do not include the action or topic.
- Search snippets that expose navigation text instead of useful summaries.

## Review checklist

- Is there one best page for the search intent?
- Does that page clearly include the query language?
- Are old or duplicate pages controlled?
- Does the search result title make sense without context?
- Does the snippet help the user decide what to click?
