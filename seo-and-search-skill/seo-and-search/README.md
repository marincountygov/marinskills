# SEO and Search Skill Package

This package contains a core skill and subskills for improving public findability through clear page titles, metadata, resident search terms, synonyms, internal search behavior, and structured data where appropriate.

## Files

- `SKILL.md` — core operating rules and workflow.
- `subskills/page-titles.md` — title tags, H1s, and search result titles.
- `subskills/meta-descriptions.md` — search descriptions and snippets.
- `subskills/resident-search-terms.md` — resident query language and search intent.
- `subskills/synonyms-and-plain-language-keywords.md` — official-to-public terminology mapping.
- `subskills/internal-search-optimization.md` — internal search result quality and synonym support.
- `subskills/structured-data.md` — schema recommendations and boundaries.

## Purpose

Use this package to make public content easier to find, understand, and select from search results. It is designed for government service pages, public information pages, notices, forms, documents, meetings, and content migrations.

## Boundaries

This package does not replace technical SEO engineering, analytics implementation, legal review, accessibility review, or content-governance review.
