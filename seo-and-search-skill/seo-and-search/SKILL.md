# SEO and Search Skill

## Purpose
Improve public findability through clear page titles, metadata, resident search terms, synonyms, snippets, internal search behavior, and structured data where appropriate.

Use this skill when creating, reviewing, rewriting, auditing, or migrating content that must be discoverable through public search engines, County site search, navigation, or search result snippets.

## Core operating principles

1. Optimize for resident intent first.
   - Use the words residents use, not only department, program, legal, or internal terminology.
   - Prefer task-oriented language: apply, pay, report, request, register, appeal, renew, contact, check, find, get help.
   - Include common synonyms when they are real user search terms.

2. Make every page's purpose explicit.
   - A search result should tell the user what the page is for before they click.
   - Page titles, headings, summaries, and metadata should align with the actual task or information need.
   - Do not optimize a page for topics it does not substantially answer.

3. Support both external and internal search.
   - External search needs clear titles, meta descriptions, canonical pages, structured data when appropriate, and durable URLs.
   - Internal search needs resident-facing labels, predictable synonyms, concise summaries, and avoidance of duplicate near-identical pages.

4. Avoid manipulative SEO.
   - Do not keyword-stuff.
   - Do not create doorway pages.
   - Do not hide text for search engines.
   - Do not prioritize rankings over accuracy, trust, or resident task completion.

5. Treat search content as public service content.
   - Search snippets should reduce uncertainty.
   - Metadata should help users choose the right page.
   - Search optimization should improve access to services, public information, and required notices.

## When to use subskills

Use the subskills as focused references:

- `page-titles.md` for title tags, H1s, and search result titles.
- `meta-descriptions.md` for summary snippets and click-decision text.
- `resident-search-terms.md` for intent-based keyword discovery and query mapping.
- `synonyms-and-plain-language-keywords.md` for aligning official terms with resident language.
- `internal-search-optimization.md` for site-search behavior, result quality, and search landing patterns.
- `structured-data.md` for schema markup recommendations and boundaries.

## Default workflow

1. Identify the page or content type.
2. State the primary resident task or question.
3. Identify the official terms and resident-facing terms.
4. Recommend or rewrite:
   - page title,
   - H1,
   - meta description,
   - search summary,
   - key synonyms,
   - internal search terms,
   - structured data if relevant.
5. Flag duplication, misleading titles, missing canonical pages, or unsupported keywords.
6. Provide a concise search-readiness checklist.

## Output patterns

For a single page, provide:

- Recommended page title
- Recommended H1
- Recommended meta description
- Primary resident search terms
- Related synonyms
- Internal search notes
- Structured data recommendation, if any
- Risks or gaps

For a collection of pages, provide:

- Search intent map
- Canonical page recommendations
- Duplicate or overlapping pages
- Title and metadata pattern
- Internal search improvements
- Measurement recommendations

## Quality checks

Before finalizing, verify that:

- The title names the actual service, topic, or task.
- The meta description accurately describes what the page provides.
- The H1 is not vague or department-only.
- Resident search terms are reflected naturally in headings or body copy.
- Synonyms support findability without creating confusion.
- Similar pages have clear differences or a canonical page plan.
- Structured data is used only when the content actually matches the schema type.

## Boundaries

This skill does not replace legal review, records review, accessibility review, design-system review, analytics implementation, or technical SEO engineering. Escalate those issues to the relevant skill or expert when they affect the recommendation.
