# Content Strategy Skill Package

This package defines a reusable content strategy skill and supporting subskills.

## Files

- `SKILL.md` — core skill and routing workflow
- `subskills/content-types.md` — content type classification and decision rules
- `subskills/audience-intent.md` — audience segmentation and user intent analysis
- `subskills/page-purpose.md` — page purpose statements and page-level decisions
- `subskills/content-lifecycle.md` — ownership, review, archival, and retirement planning
- `subskills/editorial-calendar.md` — publication and maintenance scheduling
- `subskills/content-audit.md` — audit criteria and content disposition recommendations
- `subskills/metadata-taxonomy.md` — metadata, taxonomy, findability, and governance

## Purpose

Define why content exists, who it serves, where it belongs, how long it should live, and how success is measured.

## Recommended use

Install the `content-strategy` folder as a skill package. Use the core `SKILL.md` as the routing entry point and the subskills as task-specific references.
