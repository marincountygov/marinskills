# Forms and Documents Skill Package

This package contains a core `SKILL.md` and focused subskills for improving forms, notices, letters, packets, downloadable files, and PDF-versus-HTML decisions.

## Files

- `SKILL.md` — Core routing, workflow, quality checklist, and escalation triggers
- `subskills/form-field-design.md` — Form fields, labels, grouping, validation, and required/optional logic
- `subskills/instructions-and-help-text.md` — Inline instructions, examples, helper text, and error prevention
- `subskills/notices-and-letters.md` — Notices, letters, determinations, reminders, and resident correspondence
- `subskills/pdf-vs-html-decision.md` — Format selection for HTML, online forms, PDFs, hybrids, and retirements
- `subskills/document-conversion.md` — Converting PDFs, packets, and long documents into digital-first content
- `subskills/download-page-patterns.md` — Pages that organize and explain downloadable files

## Scope

The package focuses on usability, clarity, task completion, and format decisions. It does not replace legal, records-retention, privacy, accessibility, procurement, or program-policy review.
