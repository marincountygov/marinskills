# Digital Service Design Skill Package

This package provides a core skill and subskills for improving resident task completion across County digital services.

## Files

- `SKILL.md` — core routing and service-design principles
- `subskills/service-journeys.md` — end-to-end journey mapping
- `subskills/task-flows.md` — resident task sequence design
- `subskills/forms-and-transactions.md` — forms, payments, uploads, signatures, and submissions
- `subskills/eligibility-and-requirements.md` — eligibility, requirements, and prerequisites
- `subskills/status-confirmation-next-steps.md` — confirmations, receipts, statuses, and next steps
- `subskills/failure-states.md` — errors, ineligibility, denials, missed deadlines, and recovery paths

## Purpose

Improve resident task completion for services such as applying, paying, reporting, requesting, registering, appealing, contacting, or checking eligibility.

## Suggested use

Install this folder as a skill named `digital-service-design`. Use it alongside separate skills for plain language, accessibility, content strategy, and County design standards.
