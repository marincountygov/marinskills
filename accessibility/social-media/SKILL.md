# Social Media Accessibility Skill

Use this skill to create and review accessible County of Marin social media content across platforms.

## Purpose

Social media content should be understandable, accessible, brand-consistent, and usable by people who rely on assistive technology or alternative formats.

## When to use this skill

Use this skill for:

- Social media posts.
- Post captions.
- Alt text for images.
- Hashtag review.
- Video captions.
- Graphics with embedded text.
- Emergency or public health messages.
- Event promotions.
- Multi-image posts and carousels.

## Accessibility requirements

Social media content should:

- Include alt text for informative images whenever the platform supports it.
- Avoid posting images of text as the only source of information.
- Repeat essential text from graphics in the post body or linked web page.
- Use clear, readable captions.
- Use camel case for hashtags, such as `#MarinCounty`.
- Limit emoji use and avoid using emojis as substitutes for words.
- Avoid excessive special characters, ASCII art, or decorative typography.
- Provide captions for videos.
- Provide audio description or equivalent text when visual information is essential.
- Avoid relying on color alone in graphics.
- Use sufficient color contrast in social graphics.
- Link to an accessible web page for full details when possible.

## Alt text guidance

Alt text should:

- Describe the purpose of the image.
- Include essential text shown in the image.
- Be concise but complete.
- Avoid phrases such as “image of” unless context requires it.
- Omit purely decorative details.

## Plain-language requirements

Social posts should:

- Lead with the main point.
- Include the action users need to take.
- State dates, times, locations, and deadlines clearly.
- Avoid acronyms unless they are explained.
- Use short sentences.
- Link to a source page for details.

## Brand requirements

Social content should:

- Use County-approved names, logos, colors, and tone.
- Maintain professional, service-oriented language.
- Avoid visual clutter.
- Match the seriousness of the topic.

## Review workflow

1. Identify the primary message and audience.
2. Check caption clarity and reading level.
3. Review image alt text.
4. Check video captions and audio/visual equivalents.
5. Review hashtags, emojis, links, and graphics.
6. Confirm the linked destination is accessible and current.

## Output format

When reviewing social content, provide:

- Accessibility status.
- Revised post copy if needed.
- Recommended alt text.
- Caption or transcript recommendations.
- Graphic accessibility notes.
- Brand and plain-language notes.
