# Accessibility Skills

These skills support WCAG 2.2 AA alignment and accessible digital content production.

Use `core` for shared accessibility principles, review language, severity ratings, and cross-format requirements.

Use output-specific skills for production and review workflows:
- `web`: websites, web apps, HTML, CSS, JavaScript
- `office-documents`: DOCX, PPTX, spreadsheets, memos, reports, slide decks
- `pdf`: PDF export, review, remediation, and validation
- `social-media`: social posts, images, video captions, hashtags, and platform-specific accessibility
